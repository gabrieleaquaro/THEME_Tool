Documentation

Hi, welcome to the documentation page. Here you will find all the answers to questions about how the software works.

Sections

The side menu allows you to access each section of the program. It is divided into 5 macro-areas with sub-areas:

1. Home

It's the welcome section. You will be able to access the main features of the program:

  • Report Management
  • Viewing Results
2. Current Report

This area displays the name of the current report. By clicking on it (fig.1), a text area will appear from which it is possible to rename it (fig.2).

3. Modify Report

In this area there are all the pages for the insertion and visualization of data regarding the report. The pages and their functions are described below:


Performance Shaping Factors – PSF

This page is dedicated to managing and entering values regarding the probability of error:

  • Task Description
  • Performance shaping factors

You can enter two types of values (fig.3):

  • Evaluation (menu where two or more values are present)
  • Notes (text box for entering notes associated with the previously requested value; the text area can be changed by dragging the bottom right corner).

At the end of the page there is a summary table of the values associated with each PSF(fig.4).
The page is saved automatically every time a change is made.
Each item is associated with an info button that describes it.


Direct Barriers

This page is dedicated to the management and insertion of values concerning direct barriers:

  • Safe performance of tasks
  • Compliance with safety rules/procedures
  • Safe performance in the context
  • Active participation in safety
  • Teamwork
  • Safety communication

You can enter two types of values:

  • Evaluation (menu where two or more values are present)
  • Notes (text box for entering notes associated with the previously requested value; the text area can be changed by dragging the bottom right corner).

Upon completion of a section, the average value of the answers given will appear in the bottom right-hand corner

  • (fig.5)
    • .
    At the end of the page there will be a summary table of the scores associated with each section.
    Saving the page is done automatically every time you make a change.
    Each item is associated with an info button that describes it.


    Safeguard Barriers

    This page is dedicated to the management and insertion of values regarding the safeguard barriers:

    • Non-technical Skills
    • Technical Skills
    • Safety Motivation
    • Organizational Safety Citizenship
    • Skills Assessment and Development
    • Safety Leadership
    • Safety Climate and Culture

    You can enter two types of values:

    • Evaluation (menu where two or more values are present)
    • Notes (text box for entering notes associated with the previously requested value; the text area can be changed by dragging the bottom right corner).

    Upon completion of a section, the average value of the answers given will appear at the bottom right.
    At the end of the page there will be a summary table of the scores associated with each section.
    Saving the page is done automatically every time you make a change.
    Each item is associated with an info button that describes it.


    Cultural Values

    This page is dedicated to managing and entering values regarding cultural values:

    • Individualism
    • Distance from power
    • Uncertainty Avoidance
    • Masculinity
    • Short-term orientation

    You can enter:

    • Evaluation (menu where you can enter: YES, if the statement reflects the work environment; NO, if it does not).

    Upon completion of a section, the average value of the answers given will appear at the bottom right.
    At the end of the page there will be a summary table of the scores associated with each section.
    Saving the page is done automatically every time you make a change.
    Each item is associated with an info button that describes it.


    Interventions

    This page describes the operational methods by which THEME Tool scores are associated with specific interventions.
    The status of each area can be:

    • Critical: the criticality of the element is high. By clicking on the section, a table of the elements to be performed will appear divided into: intervention area, intervention strategy, intervention description and recommendations. At the end of the section there is a text area where it is possible to insert notes (the text area can be modified by dragging the bottom right corner)
    • (fig.6)
      • .
    • Non-critical
      • : lthe criticality of the element is low. At the end of the section there is a text area where it is possible to insert notes (the text area can be modified by dragging the bottom right corner).

    Saving the page is done automatically every time you make a change.


    Results

    This page generates graphs for the values described on the following pages: error probability, direct barriers, safeguard barriers, and cultural values.
    Each page is associated with a table and two charts of type "Radar" (fig.8) and "Bars" (fig.7).

    At the bottom of the page is a bar graph and a summary table of all values.

    To better visualize the single graphs, click on the icon at the top right of each section. To download the single chart it is necessary to click on the icon at the top right of each section. To download a pdf file in which is present the entire page it is necessary to click on the icon on the left of the section Results (fig.8)

    4. Report History

    In this area there is the archive entry from which it is possible to access the report history. The pages and their functions are described below:

    Archive

    In this page there is the history of all reports in date order.
    To select a report as the current report you need to click on the button Open(fig.9).
    To delete a report it is necessary to click on the button Delete(fig.9).

    To search for a specific report enter the name in the search bar at the top. (fig.10)

    4.Reports Save

    In this area there are 3 functionalities:

    • New Report: By selecting this item (fig.11) a popup will appear where you can enter the name of the new report and proceed to its creation;(fig.12)
    • Download Report:by selecting this item the current report will be saved in the desired folder;
    • Load Report: by selecting this item you can load a report from a folder and set it as the current report.
